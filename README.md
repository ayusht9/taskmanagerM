
# Task Manager

- A simple **Task Manager** application built using **Node.js, Express, MongoDB, and React**.
- Applied **CRUD operations**

## Features

1. Create new tasks
2. View all tasks
3. Mark tasks as completed
4. Update tasks (PUT / PATCH)
5. Delete Task
6. MongoDB persistence
7. React frontend
8. RESTful API design

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* dotenv
* cors

### Frontend

* React
* Fetch API
* CSS

## Endpoints

### Create Task

```http
POST /tasks
```

### Get All Tasks

```http
GET /tasks
```

### Get Task By ID

```http
GET /tasks/:id
```

### Update Task (PUT)

```http
PUT /tasks/:id
```

### Update Task (PATCH)

```http
PATCH /tasks/:id
```

### Delete Task

```http
DELETE /tasks/:id
```
